# @backoffice/constants

Shared Constants for Backoffice use

## Description

This package exports shared constants for use within Backoffice, an open-source headless CMS.

For more information about Backoffice, visit the [official website](https://backoffice.io).

## Installation

```shell
npm install @backoffice/constants
```
