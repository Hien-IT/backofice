# `@backoffice/stores`

Shared Backoffice Studio state for use in components, extensions, and the `@backoffice/app` routes. Stores are
[Pinia](https://www.npmjs.com/package/pinia)-based stores.

## Installation

```
pnpm add @backoffice/stores
```

## Usage

```ts
import { useAppStore } from '@backoffice/stores';

const appStore = useAppStore();
```
