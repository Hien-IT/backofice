# @backoffice/composables

Shared Vue composables for Backoffice use

## Description

This package provides shared Vue composables for use within Backoffice, an open-source headless CMS.

For more information about Backoffice, visit the [official website](https://backoffice.io).

## Installation

```shell
npm install @backoffice/composables
```
