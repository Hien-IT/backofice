# @backoffice/extensions-sdk

A toolkit to develop extensions to extend Backoffice

## Description

This package is a toolkit to help developers create extensions for use in Backoffice, an open-source headless CMS.

For more information about Backoffice, visit the [official website](https://backoffice.io).
